This repository is for reporting issues with **Lubricant**, a game modification for *Minecraft: Java Edition*, adding additional in-game content. This project is developed by **SeaSerpant1040**, with occasional contributions from **Apocalypsing**

Lubricant requires access to a copy of the *Java Edition* release of *Minecraft*; it is not compatible with the *Bedrock Edition* releases of the game. Additionally, *Fabric Loader*, and *Fabric API* are also required to be installed to use Lubricant.
